vCardLib Documentation

Support for vCard version 3.0 and for saving vCards have been added

vCardLib is a .NET library written entirely in C#. It is designed to enable developers read contacts stored in a vCard file (*.vcf or *.vcard). As at today 10th September 2016, the latest version of the library (v2.1.0) supports vCard files of version 2.1 and 3.0 only. The library has its classes under the namespace vCardLib. The class 'vCard' has static methods FromFile and FromStreamReader which as their names imply allow you extract contacts from a file and a StreamReader object respectively.

The FromFile and FromStreamReader methods return a vCardCollection object which inherits from CollectionBase. The vCardCollection object implements IEnumerable and can be iterated over using the foreach operator. Each object is of the vCard type from which properties can be read.

Library Version 2.1.0 Additional Properties

With the release of version 2.1.0 of this library, support has been added for writing of vCard objects to vcf files. It supports writing one file and of writing a vCardCollection to a file. To make use of the writing features of the library, see some exaple code below. 

        
          vCard vcard = new vCard();
          //add some properties
          vcard.Version = 2.1F;
          vcard.Surname = "John";
          vcard.Firstname = "Smith";
          PhoneNumber number  = new PhoneNumber();
          number.Number = "09889302222";
          number.Type = PhoneNumberType.Cell;
          vcard.PhoneNumbers.Add(number);

          string filePath = "path-to-vcf-file";
          //To write a single vcard in its default version
          vcard.Save(filePath, [Optional] WriteOption.ThrowError);
          //To write a single vcard in another version
          vcard.Save(filePath, 3.0F, [Optional] WriteOptions.Overwrite);

          vCardCollection collection = new vCardCollection();
          collection.Add(vcard);
          //To write a vcard collection in their default versions
          collection.Save(filePath, [Optional] WriteOption.ThrowError);
          //To write a vcard collection in another version
          collection.Save(filePath, 3.0F, [Optional] WriteOptions.Overwrite);

Library Version 2.0.0 Additional Properties

  1. PhoneNumbers: PhoneNumberCollection
    • Type: PhoneNumberType (None, BBS, Modem, Car, ISDN)
  2. EmailAddresses: EmailAddressCollection
    • Type: EmailAddressType (None)
  3. Addresses: AddressCollection
    • Type: AddressType (None, Domestic, Postal, Parcel, International)
  4. TimeZone: string

Library Version 1.1.3 Additional Properties

  1. Pictures: PhotoCollection Support has been added for JPEG and GIF image URLs and for JPEG images. Support for other formats coming soon.
    • Picture: Bitmap
    • Type: PhotoType (Image, URL)
    • Encoding: PhotoEncoding (JPEG, GIF)
    • PhotoURL: string
  2. Gender: GenderType (Male, Female, Other, None) None is the default if no gender specified
