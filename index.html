vCardLib Documentation

vCardLib is a .NET library written entirely in C#. It is designed to enable developers read contacts stored in a vCard file (*.vcf or *.vcard). As at today 11th June 2016, the latest version of the library (v2.0.0) supports vCard files of version 2.1 and 3.0 only. The library has its classes under the namespace 'vCardLib'. The class 'vCard' has static methods 'FromFile' and 'FromStreamReader' which as their names imply allow you extract contacts from a file and a StreamReader object respectively.


The 'FromFile' and 'FromStreamReader' methods return a vCardCollection object which inherits from 'CollectionBase'. The 'vCardCollection' object implements 'IEnumerable' and can be iterated over using the 'foreach' operator. Each object is a vCard object from which properties can be read.

Library Version 2.0.0 Property Changelog

Support for vCard version 3.0 has been added

  1. PhoneNumbers: PhoneNumberCollection
    • Type: PhoneNumberType (None, BBS, Modem, Car, ISDN)
  2. EmailAddresses: EmailAddressCollection
    • Type: EmailAddressType (None)
  3. Addresses: AddressCollection
    • Type: AddressType (None, Domestic, Postal, Parcel, International)
  4. TimeZone: string

Library Version 1.1.3 Property Changelog

  1. Pictures: PhotoCollection Support has been added for JPEG and GIF image URLs and for JPEG images. Support for other formats coming soon.
    • Picture: Bitmap
    • Type: PhotoType (Image, URL)
    • Encoding: PhotoEncoding (JPEG, GIF)
    • PhotoURL: string
  2. Gender: GenderType (Male, Female, Other, None) None is the default if no gender specified
