vCardLib Documentation

vCardLib is a .NET library written entirely in C#. It is designed to enable developers read contacts stored in a vCard file (*.vcf or *.vcard). As at today 3rd June 2016, the latest version of the library (v1.1.2) supports vCard files of version 2.1 only. The library has its classes under the namespace 'vCardLib'. The class 'vCard' has static methods 'FromFile' and 'FromStreamReader' which as their names imply allow you extract contacts from a file and a StreamReader object respectively.


The 'FromFile' and 'FromStreamReader' methods return a vCardCollection object which inherits from 'CollectionBase'. The 'vCardCollection' object implements 'IEnumerable' and can be iterated over using the 'foreach' operator. Each object is a vCard object from which properties can be read.
